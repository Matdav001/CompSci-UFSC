# :computer: CompSci-UFSC

This repository documents my journey through the Computer Science program at **UFSC** (Universidade Federal de Santa Catarina).

[![UFSC](https://img.shields.io/badge/University-UFSC-blue)](https://ufsc.br/)
[![Course](https://img.shields.io/badge/Degree-Computer%20Science-green)](https://cco.ufsc.br/)

It contains:
- :books: Course notes
- :pencil2: Exercises and assignments
- :pencil: Exams and study material
- :computer: Course projects

The goal is to organize my learning progress and keep all course material structured by semester.

## :open_file_folder: Repository Structure

Each folder represents a semester and contains the courses planned for that term, even if some were completed in a later semester.

